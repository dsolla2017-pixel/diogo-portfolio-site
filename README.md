## Sobre o Projeto

O CESOA Forward é uma proposta de transformação digital para o crédito habitacional da CAIXA Econômica Federal. O projeto apresenta uma arquitetura completa baseada em Clean Architecture, Inteligência Artificial e governança de dados com Pipeline Medallion.
